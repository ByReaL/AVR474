bool BATTVMON_IsCellVoltageTooHigh ( vadcIndex_t  cell  ) 

Investigate if any of the battery cells are too high.

Requires that a V-ADC scan has been run.

Parameters:
cell Which cell to check. Is not checked by this function, but readCellVoltage will return 0 if the cell is invalid (and thus this function will always return false in that case)
Returns:
True is returned if any of the cells have too high voltage, otherwise false.
