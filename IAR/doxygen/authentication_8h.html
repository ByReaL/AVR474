Enumeration Type Documentation

enum AUTH_Status

Enumerator:
AUTH_Status_Start  Should be the value of the status flag when sent from host, and is returned if the battery hasn't started the authentication algorithm yet.
AUTH_Status_Executing  Status during execution
AUTH_Status_Finished  The authentication algorithm is finished and the data ready to send to host
AUTH_Status_Error  Status if any error occurs (depends on crypto used)

Function Documentation

void AUTH_Execute ( SBS_command_t *  sbsCmd  ) 

Run the chosen authentication algorithm.

Runs the authentication algorithm on the data in sbsCmd and stores the result in authenticationResponse to be retrieved by AUTH_GetResponse().

The data in sbsCmd will be destroyed

Only runs the algorithm if the status byte is AUTH_Status_Start.

Parameters:
sbsCmd Pointer to SBS command parameters
